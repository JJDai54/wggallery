Hilfe: wgGallery

Beschreibung

Das wgGallery-Modul kann zur Präsentation Ihrer Bilder in Alben auf Ihrem XOOPS verwendet werden.

Installieren/Deinstallieren

Es sind keine speziellen Maßnahmen erforderlich, führen Sie den Standardinstallationsprozess aus und extrahieren Sie den Ordner wggallery in das Module-Verzeichnis. Installieren Sie das Modul über Admininstration -> System -> Module.

Wenn Sie zustätzliche Informationen benötigen, lesen Sie die Installationshinweise unter XOOPS Operations Manual

Funktionen

Das Modul wird laufend weiterentwickelt, um die Anforderungen auch in Zukunft erfüllen zu können.
Aus diesem Grund sind alle Entwickler eingeladen, Verbesserungs- oder Änderungsvorschläge auf github einzusenden.
