Aide : wgGallery

Description

le module wgGallery peut être utilisé pour présenter vos images dans des albums au sein de votre XOOPS

Installation / Désinstallation

Aucune mesure particulière n'est nécessaire, suivez le processus d'installation standard et extrayez le dossier wggallery dans le répertoire ../modules. Installez le module via Admin -> System Module -> Modules.

Des instructions détaillées sur l'installation des modules sont disponibles dans le Manuel d'exploitation de Xoops

Caractéristiques

Le module wgGallery continue de se développer, pour arriver aux conditions de création de modules, de plus en plus sophistiquées.
Pour cette raison, j'invite tous les développeurs à signaler et à envoyer sur github tout changement ou ajout à ce module, afin que nous puissions contribuer conjointement au développement
